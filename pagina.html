Etiquetas HTML

HTML utiliza una serie de etiquetas para estructurar el contenido de las páginas web, permitiendo a los desarrolladores crear documentos ricos y bien organizados. A continuación, exploraremos algunas de las etiquetas más fundamentales:



angeld


<body>

Define el contenido principal de un documento HTML

<header>

Define el encabezado de una sección o documento

<aside>

Define contenido aparte del contenido principal

<h1>

Define el encabezado más importante

<p>

Define un párrafo de texto

<nav>

Define una sección de navegación en el documento

<a>

Define un hipervínculo

<img>

Inserta una imagen

<ul>

Define una lista desordenada

<ol>

Define una lista ordenada

<li>

Define un elemento de lista

<div>

Define una división o sección en un documento

<footer>

Define el pie de página de una sección o documento
